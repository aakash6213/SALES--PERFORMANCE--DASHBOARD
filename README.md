# Sales Performance Dashboard

## 📌 Project Overview

The Sales Performance Dashboard is an interactive Power BI project designed to analyze sales performance across different regions, product categories, customer segments, and shipping modes. The dashboard provides key business insights through interactive visualizations and KPI tracking, enabling data-driven decision-making.

---

## 🛠️ Tech Stack

- Power BI
- Microsoft Excel
- Power Query
- DAX

---

## 📊 Key Performance Indicators (KPIs)

- **Total Sales:** $2.30M
- **Total Profit:** $286.40K
- **Total Orders:** 5K
- **Total Quantity Sold:** 38K

---

## 📈 Dashboard Features

- Sales Analysis by Category
- Profit Analysis by Category
- Regional Sales Performance
- Customer Segment Analysis
- Ship Mode Analysis
- Monthly Sales Trend
- Interactive Filters and Slicers
- KPI Cards

---

## 🔍 Business Insights

- Identified top-performing product categories based on revenue and profit.
- Analyzed regional sales performance to determine high- and low-performing markets.
- Compared customer segments to understand purchasing behavior.
- Evaluated the impact of different shipping modes on overall sales.
- Monitored key business metrics through an interactive dashboard for better decision-making.
